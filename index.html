Responsive Type Tester

It's difficult to quickly iterate on how type and margins respond to screen size in Figma. This tool is designed to make that process a little easier, with a variety of common typographic layouts set up to assess type and margin sizing.

Testing in-browser also gets closer to the real set of options available in production: viewport units and calc() formulations are all likewise difficult to specify and fine-tune in flat layouts.
